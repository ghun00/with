현재 학생 상세 페이지의 To Do 기능을 ‘활동 관리’ 기능으로 변경하고자 합니다.

기존처럼 단순히 할 일을 등록하고 완료 여부만 확인하는 구조가 아니라, 학생이 현재 어떤 활동을 준비하고 있고 어떻게 진행하고 있는지 지속적으로 기록·관리하는 기능으로 전환이 필요합니다.

주요 변경 방향
To Do 탭을 활동 관리 탭으로 변경
학생/컨설턴트 To Do 구분 제거
모든 활동은 해당 학생을 기준으로 등록 및 관리
활동 상태를 아래 3가지로 구분
진행 예정
진행 중
활동 완료
현재 진행 중인 활동과 완료한 활동을 구분하여 목록 제공
단순 체크리스트가 아니라, 활동별 상세 내용을 하나의 문서처럼 지속해서 수정·관리할 수 있는 구조로 적용
활동 등록 항목
활동명
현재 상태
활동 분류
마감일
활동 세부 내용
Sub-task 목록
유저 플로우

활동 생성 버튼 클릭
→ 활동 정보 및 세부 내용 작성
→ 활동 생성
→ 활동 상세 화면에서 내용, 상태, 마감일 및 Sub-task 지속 수정
→ 진행 과정 및 변경 내역 확인
→ 활동 완료 처리

디자인과 UX는 flex 워크플로우https://flex.team/landing/service/workflow를 참고하여, 활동 목록 → 활동 상세 문서 → 진행 상태 및 변경 이력 확인 흐름이 직관적으로 이어지도록 구성 부탁드립니다.

특히 활동 상세 화면에서는 단순한 입력값만 보여주는 것이 아니라, 상태 변경과 내용 수정 등 활동의 진행 과정을 시간순으로 확인할 수 있는 UX가 필요합니다.
